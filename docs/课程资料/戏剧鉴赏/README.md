### 黄可、李志军、孙美玲

**老师超级超级好，讲的东西也有很多自己的思考！每次上课前都会在学习通发通知，每次都不一样但都很有趣！！！期末是一张卷子自己拿回去写！无脑冲就对了！！！：**
