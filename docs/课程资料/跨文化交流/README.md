**Unnamed: 4：**

签到：20分
缺席一次扣2分
演讲：20分  【2-3人为一组
每组20-25分钟：演讲（15分钟）+答疑（5-10分钟）
自选题目：国家、电影、书籍、艺术……
评估标准：
是否能合理运用课堂所学理论？
内容是否有条理？
演讲是否清晰流畅？
能否准确回答听众提问？
时间控制是否得当？】
论文：50分【整合演讲内容（可根据听众提问加以改进）
每组提交一篇
字数：>3000字
查重率：<30%
格式：标题+摘要+关键词+正文+参考文献
评分标准：
内容：是否正确运用理论，论证是否充分，……
格式：是否符合规范

】
课堂参与：10分
发言一次加2-5分
